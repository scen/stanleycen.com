About this site

This website is an open-source Sinatra web application running on Ruby 1.9.3 and lives somewhere in the Heroku cloud.

It's powered by lots of awesome things made by amazing people:

  • jQuery
  • Sass
  • Fancybox
  • Sinatra
  • Splash images from unsplash

This is my latest (probably 6th) redesign of the site, featuring parallax scrolling and responsive design!

The design was pushed out on November 19th 2013. Let me know of any bugs you find!
